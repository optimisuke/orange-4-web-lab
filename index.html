キーパッドでプログラミング
  1. RST リセット
  2. ADR → キーパッドで2桁入力してアドレスを指定
  3. キーパッドで1桁ずつ入力 → RAM に書き込み、次のアドレスへ自動移動
  4. INC 書き込まずに次のアドレスへ移動
  5. RUN 実行
